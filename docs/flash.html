🧵 SpoolEase
Filament inventory & spool management for Bambu Lab printers loading…

Choose install type

🔄 Upgrade recommended

Keeps your Wi-Fi credentials, printer settings and spool database intact.

  • Connect SpoolEase via USB
  • Click Upgrade and select the serial port
  • Wait ~30 s, then power‑cycle the device

🆕 Fresh install

Erases all settings. The browser will guide you through Wi-Fi setup via Improv Wi-Fi immediately after flashing — no need to connect to a separate access point.

  • Connect SpoolEase via USB
  • Click Fresh install and select the serial port
  • Confirm the erase when prompted
  • Enter your Wi-Fi credentials in the browser when asked
  • Then open the SpoolEase config page to add your printer
⚠️ Your browser does not support Web Serial. Please use Chrome or Edge (desktop).

Requires Chrome or Edge on desktop. Not supported on Firefox, Safari, or mobile. Browser compatibility ↗
