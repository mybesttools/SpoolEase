This page hosts static web assets for the SpoolEase firmware project.

Assets

  • translate.js – Inventory editor UI translator (Polish + future languages)

Adding a translation

Edit docs/translate.js in the repository and add a new key under TRANSLATIONS:

TRANSLATIONS = {
  pl: { 'Filters': 'Filtry', ... },
  de: { 'Filters': 'Filter', ... },  // ← new language
};

Commit and push to main; GitHub Actions will deploy automatically.
