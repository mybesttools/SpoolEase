Web Installer

🧵 Flash SpoolEase firmware →

Uses the Web Serial API (Chrome / Edge desktop) to flash firmware directly from the browser. No software installation needed.

Firmware is built automatically by GitHub Actions on every push to main and deployed here. The pipeline:

  1. Installs the Xtensa ESP32-S3 Rust toolchain via espup
  2. Runs cargo build --release in core/
  3. Runs espflash save-image --merge to produce a single flat binary (SpoolEase-flash.bin at offset 0x0000)
  4. Stamps the version from Cargo.toml into docs/firmware/manifest.json
  5. Deploys docs/ (including the freshly built binary) to GitHub Pages

No manual binary copying required. Just push code changes to main.

Adding a language

Translations are embedded directly in the firmware HTML (see core/src/web_app.rs — build_inventory_html() and translate_script()). To add a new language, add its translation table there, add it to the language selector bar and /api/language whitelist, then rebuild and flash.
